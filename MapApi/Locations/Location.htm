地图提供者（MapProvider）

名称 显示名 类型 长度 精度 主键 允许空 备注
Id 编号 Int32 AI N
Name 名称 String 50
Kind 提供者 Int32 N
Enable 启用 Boolean N
Server 服务地址 String 50
AppKey 应用密钥 String 500 多个密钥逗号隔开
CreateUser 创建者 String 50
CreateUserID 创建用户 Int32 N
CreateIP 创建地址 String 50
CreateTime 创建时间 DateTime N
UpdateUser 更新者 String 50
UpdateUserID 更新用户 Int32 N
UpdateIP 更新地址 String 50
UpdateTime 更新时间 DateTime N
Remark 备注 String 500
